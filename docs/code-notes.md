# Ummiby Code Notes

## Explore links format

Use this when a duaa has extra resources:

```js
explore: [
  {
    type: "audio",
    title: "Listen to the Duaa",
    url: "https://real-link-here"
  },
  {
    type: "article",
    title: "Read an Explanation",
    url: "https://real-link-here"
  },
  {
    type: "video",
    title: "Watch a Lecture",
    url: "https://real-link-here"
  }
]
